基本

Selection

css 互換セレクターで要素を選択

d3.select('body').selectAll('div');

Data Binding

Selection に対して data を bind する。


var data = [4, 3, 5, 6, 10, 14];

d3.select('body').selectAll('div')
    .data(data); // selection に data を bind

Data-Driven Update

data を使って selection を update する


data = [4, 3, 5, 6, 10, 14]

d3.select('body').selectAll('div')
    .data(data)
    .style('height', 20)
    .style('width', (d) -> d)
    .attr('class', 'bar')
